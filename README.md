# Urban Ladies PRO

Sistema integral de gestión para salones de belleza. Aplicación web profesional, moderna y escalable.

## 📋 Descripción

Urban Ladies PRO es una solución completa para gestionar operaciones de salones de belleza, incluyendo:
- Gestión de citas
- Gestión de clientes
- Dashboard ejecutivo con KPIs
- Catálogo de servicios
- Reportes y análisis
- Autenticación y control de acceso

## 🛠️ Tech Stack

- **Frontend**: React 18+ con TypeScript, Tailwind CSS, Shadcn/ui
- **Backend**: Node.js/Express con TypeScript
- **Base de datos**: PostgreSQL
- **Autenticación**: JWT
- **Gráficos**: Recharts
- **Reportes**: PDFKit, xlsx

## 🚀 Instalación y Configuración

### Requisitos
- Node.js 18+
- PostgreSQL 14+
- npm o yarn

### Backend Setup

```bash
cd backend
npm install
cp .env.example .env
npm run dev
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```
